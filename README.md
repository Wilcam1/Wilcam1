### Sobre mí
Soy desarrollador de software con enfoque en el desarrollo web y la aplicación práctica de la inteligencia artificial. Me especializo en crear herramientas interactivas que solucionan problemas reales, combinando la lógica de backend (como Python y Flask) con modelos predictivos de Machine Learning y visualización geoespacial interactiva.

Trabajo con un objetivo claro: diseñar interfaces de usuario premium y limpias (*clean code*), optimizar el procesamiento de datos y asegurar un rendimiento óptimo de la infraestructura web.

### Proyectos destacados

#### 🚗 [Viaja Inteligente](https://github.com/Wilcam1/Viaja_Inteligente)
Aplicación web interactiva que permite a los conductores en Colombia planificar sus rutas terrestres, estimar costos y optimizar el consumo de combustible de sus vehículos mediante un modelo de IA.
* **Características:** Geocodificación interactiva, mapas oscuros premium, filtros de puntos de interés (POIs) espaciales (peajes, gasolineras, hoteles) y cálculo de ahorro.
* **Tecnologías:** Flask, scikit-learn, Leaflet.js, OpenStreetMap API.
